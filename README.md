# PDF 演示工具 - Vue 3 版本

基于 Vue 3 + Vite 重构的 PDF 演示工具，支持绘图标注、章节导航和全屏演示。

🔗 **在线演示**：https://ppt.yangkunlong.top

## 技术栈

- **Vue 3** - 组合式 API
- **Vite** - 现代化构建工具
- **Pinia** - 状态管理
- **PDF.js** - PDF 渲染引擎

## 项目结构

```
src/
├── components/
│   ├── UploadScreen.vue     # 文件上传界面
│   ├── Sidebar.vue          # 缩略图侧边栏（支持章节分组）
│   ├── Toolbar.vue          # 工具栏
│   ├── SlideCanvas.vue      # 主画布（PDF + 绘图层）
│   ├── FloatingToolbar.vue  # 浮动工具栏（智能定位）
│   ├── OutlinePanel.vue     # 目录面板
│   └── OutlineItem.vue      # 目录项组件
├── stores/
│   └── pdf.js               # PDF 状态管理
├── App.vue                  # 根组件
├── main.js                  # 入口文件
└── style.css                # 全局样式
```

## 快捷键

### 导航
- `← / →` - 上一页 / 下一页
- `↑ / ↓` - **上一章 / 下一章**
- `Home / End` - 首页 / 末页
- `Space` - 下一页

### 工具
- `F5` - 全屏演示
- `P` - 画笔工具
- `E` - 橡皮擦工具
- `L` - 激光笔
- `C` - 清空当前页绘图
- `Ctrl+Z` - 撤销
- `ESC` - 退出全屏

## 新功能亮点

### 1. 章节分组缩略图
- 自动识别 PDF 大纲/书签
- 按章节分组显示缩略图
- 点击章节标题可折叠/展开

- 可拖拽到任意位置
- 自动判断展开方向（向上/向下）

### 3. 章节导航
- 上下方向键快速跳转章节
- 自动跳转到章节起始页
- 无章节时等同于翻页

### 4. 全屏演示优化
- 进入全屏自动激活激光笔
- 退出全屏自动取消工具选择
- 画笔粗细默认中值（更合适的初始值）

## 迁移说明

从原生 JS 迁移到 Vue 3 的主要改进：

1. **组件化架构** - 每个功能模块独立为 Vue 组件，代码更清晰
2. **响应式状态** - 使用 Pinia 管理全局状态，自动更新 UI
3. **更好的开发体验** - Vite 提供热更新，开发效率更高
4. **章节功能** - 新增 PDF 大纲识别和章节导航
5. **智能 UI** - 浮动工具栏自适应定位
6. **易于维护** - 组件职责单一，便于测试和扩展

原始文件已备份为 `app.js.old` 和 `index.html.old`。

## 技术细节

### PDF.js 集成
- 使用 `shallowRef` 避免 Vue 响应式包装破坏 PDF.js 私有字段
- 渲染任务取消机制防止多次渲染冲突
- 通过 script 标签加载 PDF.js 避免 ESM 模块问题

### 性能优化
- 缩略图按需渲染
- 折叠章节时保留渲染状态
- 展开时自动重新渲染缩略图
